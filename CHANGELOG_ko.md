# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-01-15

### Added
- **Smart Padding Control**: Added `padding` option to automatically add space around text.
- **Improved Bounding Box Calculation**: Used ImageMagick trim and offset logic to ensure text is perfectly framed.
- **Bilingual Documentation**: Added `_ko.md` versions for all main documentation files.

## [1.0.0] - 2026-01-15

### Changed
- Renamed gem to `text2image`.
- Initial stable release.
